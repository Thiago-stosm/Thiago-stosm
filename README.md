## 👋 Olá, eu sou o Thiago!

## Sobre mim

Sou estudante de Ciência da Computação no Instituto Mauá de Tecnologia (São Caetano do Sul – SP), com foco em desenvolvimento backend e análise de dados. Tenho interesse em construção de APIs robustas com Java e Spring Boot, modelagem de banco de dados relacional e boas práticas de arquitetura de software.

Busco evoluir constantemente em fundamentos de engenharia de software, performance, estrutura de dados e integração entre sistemas, conectando teoria acadêmica com aplicação prática em projetos reais.
